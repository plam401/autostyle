autostyle 根据实际的项目需求产生，在很多页面中，由于需求的变动，在项目维护中，需要频繁的修改页面的元素宽度，高度，外边距，内边距。所以我个人制作了这个可以在标签行内简单的调整宽高和边距数值的js组件。方便在维护中修改宽高数值。基于jquery开发。
使用的方法是:列如，w='100';就会在页面head标签中生成一个style标签，生成一个.w100{width:100px}的样式，然后把此标签引用一个class=‘w100’的样式，没有直接用jquery的css方法修改样式，是考虑到seo。这样的页面相对比较友好，不会对页面的收录有影响。
如果觉得需要可以下载后使用在项目维护中，简单好用。
